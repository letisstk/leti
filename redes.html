Redes de computadores

Uma rede de computadores é um conjunto de equipamentos interligados de maneira a trocarem informações e compartilhem recursos, como arquivos de dados gravados, impressoas, modems, softwares e outros equipamentos.

Importância das redes de computadores

Proporcionam economia, compartilhamento de recursos, como e impressoras e projetores, bem como de serviços, que é o caso da internet. Permite também, por meio do compartilhamento de arquivis, o trabalho em equipe dos colaboradores.

LAN WAN

Lan (Local Area Network) é uma rede de computadores que cobre uma área limitada, geralmente um edifício ou um campus. É usada para conectar dispositivos em uma espaço físico limitado, como uma casa, escritório ou sala de aula. Já Wan (Wide Area Network) é uma rede de computadores que abrane uma área geográfica maior, geralmente de uma cidade, um estado ou até mesmo um país. Ela é usada para conectar dispositivos que estão fisicamente distantes entre si.
